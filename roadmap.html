Roadmap

LIVRE OS is being developed in stages, starting from core specifications and a reference wallet, then expanding into developer tools and institutional integrations.

Phase 0 — Specification

  • Solivre Kernel spec
  • LivreID namespace standard
  • Proof Engine & LivreVault design
  • Key management & node model

Phase 1 — MVP

  • Reference Solivre wallet
  • Basic LivreID registration
  • Single proof type (e.g. AGE_OVER_18)
  • GC/MPC integration with COTI nodes

Phase 2 — Developer Platform

  • LivreKit SDKs and REST APIs
  • Integration guides for companies
  • More proof types and claims

Phase 3 — Ecosystem

  • Institution partnerships and pilots
  • Open node participation
  • Expanded governance and upgrades
