Roadmap

A living path from local demos to a sovereign, networked identity layer.

  1. M1

    Define MVP Scope

    Lock the minimal feature set for the Solivre Identity Node, Agent, and Verifier — plus their API contracts.

    Area: Solivre · Status: In Progress

  2. M2

    Solivre Kernel Spec

    Finalize the Identity Kernel interfaces used by apps, agents, and proof orchestrators.

    Area: Solivre · Status: Planned

  3. M3

    LivreID Namespace Draft

    Design a human-friendly namespace for identities and apps, including reservation, resolution, and portability.

    Area: Namespace · Status: Planned

  4. M4

    Proof Request Flow v0.1

    Implement the end-to-end flow: Institution → Proof Request → Solivre Agent → Proof → Verifier.

    Area: Proof Engine · Status: Planned

  5. M5

    Smart Contract Interfaces v1

    Expose proof verification and claim references on-chain, so other protocols can consume LIVRE OS signals.

    Area: Contracts · Status: Planned

This roadmap is intentionally small. The goal is to sharpen the core experience before scaling to a full network.
