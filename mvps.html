Solivre MVP Suite

A set of focused prototypes that exercise the LIVRE OS design in the real world.

Identity Node

Live

One Fastify process that serves the Agent & Verifier UIs, exposes APIs, and keeps encrypted identities in a local Vault.

  • Stores identity records, attributes, and commitments.
  • Generates deterministic demo proofs for age + residency templates.
  • Hosts Agent and Verifier on the same origin for easy demos.

Run the node locally and open the Agent & Verifier consoles.

Agent Console

Live

A browser console where a human user (the Agent) can create identities, attach attributes, and request proofs.

Identity creation

Create multiple local identities and switch between them via a dropdown.

Attributes & Vault

Bind birthdate + country to identities and write them to the encrypted Vault.

Proof export

Generate proofs and export Identity + Proof JSON for verification.

This console will evolve into a full Solivre Agent with multiple proof templates, notifications, and app-specific proof flows.

Verifier Console

Live

A simple UI that lets integrators paste an Identity payload and a Proof bundle, and see whether the claim checks out.

  1. Paste the Identity JSON exported from the Agent.
  2. Paste the Proof JSON (or entire bundle).
  3. Click Verify to see whether the proof matches the stored attributes + template.

Intended role

In production, Verifiers will usually be apps or contracts; this console is a human-friendly debugging surface.
